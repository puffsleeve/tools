# ぱふす。TOOLS

配信・動画制作・画像制作用に自作しているツール置き場です。すべてブラウザだけで動作し、ファイルをサーバーにアップロードすることはありません。制作は AI と共同で行っています。

🔗 公開ページ: https://puffsleeve.github.io/tools/

## ツール一覧

### 縦型レイアウト変換ツール（`tools-shorts-vertical.html`）
横型動画を YouTube Shorts 用の 9:16 レイアウトに変換します。タイトル・動画・イラストのサイズや位置を自由に調整でき、WebM 形式で書き出せます。

### フレーム合成ジェネレーター（`tools-frame-generator.html`）
フレーム・イラスト・背景の画像を重ねて1枚の PNG に合成します。書き出しサイズはフレーム画像のサイズに自動で合わせられます。

## 特徴

- すべての処理がブラウザ内で完結（画像・動画はどこにも送信されません）
- インストール不要、サーバーサイド不要
- 推奨環境: Google Chrome（デスクトップ／モバイル）。Safari など他ブラウザでは一部機能が正しく動作しない場合があります

## 使い方

基本的には公開ページ（GitHub Pages）にアクセスして使用してください。

もし公開ページが開けない・重い・動かないなどの場合は、このリポジトリの HTML ファイルをダウンロードして、自分のPC上でブラウザで直接開いても同じように使えます。
