Local Video Tool — No Upload, No Server

横型動画を縦型ショートにレイアウト。

TITLE VIDEO ILLUST
タイトル・動画・イラストのサイズや位置を自由に調整できます。
推奨環境：Google Chrome（デスクトップ／モバイル）。他のブラウザ（特にSafari）では、フォントの表示や書き出し機能が一部正しく動作しない場合があります。
01 / 動画を選ぶ
クリック または ドラッグ＆ドロップ 横型の動画（mp4 / mov / webm）
02 / 動画のサイズ・位置
初期値は横幅ぴったり。動画を読み込むと、画面の縦幅いっぱいまで拡大できるように上限が自動で広がります。
縦位置の初期値（320px）はiPhoneのインカメラを避けた位置です。
外枠の線
03 / タイトル フォント
フォントの読み込みには時間がかかることがあります。切り替え直後はプレビューが一瞬崩れて見える場合があります。
スタイル
横位置
04 / 背景
クリック または ドラッグ＆ドロップ 背景用の静止画（1080×1920px 推奨）
05 / イラスト（左下）
クリック または ドラッグ＆ドロップ 静止画（png / jpg）
画像を読み込んだ時点の動画のサイズ・位置をもとに、動画に重ならない最大サイズが自動計算され、初期位置として左下に配置されます（この初期計算はその時だけ行われます）。そこから上のスライダーでサイズ・位置を自由に調整できます。動画をあとから動かしても、イラストの位置は追従しません。
1080×1920pxの画像をアップロードした場合のみ、動画の上に重なる全面フレームとして配置されます（この場合サイズ・位置の調整は無効）。
プレビュー & 書き出し
動画の外側に赤いフレームはつきません。プレビューの範囲を定めるものです。
◆ すべての処理はこの端末上で完結（動画はサーバーに送信されません） / 書き出し形式：WebM (VP9/VP8) ◆
